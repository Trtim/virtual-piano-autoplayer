# virtual-piano-autoplayer
Autoplayer made in Python for Virtual Piano (Roblox or Website)


Steps to use it:
1) Download the "autoplayer.py" file
2) Have python installed (Honestly any version works)
3) In the script you can edit the sheets I have with any other sheets you got just change whatevers inside
4) Press "[" or "]" to start it up and continue pressing it for the song to continue. It basically just plays the notes for you but you still need to be on rhythm
5) press "=" to exit the script and you can restart the process
